Digital Library

Q&As

What is Digital Library project about?

Digital Library is a website for a public library that is owned and funded by country so there's no need to pay for books. It was made to ease people's access to books and to improve work experience for employees.

How do I use it?

You can simply open catalog and browse it. Or you can register and be able to order paper books, download electronic books and listen to audio ones, unless you're library's employee.

Users and their abilities

There are 4 types of users: Unregistered, Registered, Administartor, Manager. Everyone has different abilities.

As an Unregistered user you will be able to...

See books' details

Search for books

Filter books by type, genre, publisher

Register

As a Manager you will be able to...

do everything unregistered user can and...

Add, edit, delete genres

Add, edit, delete books.

To create book you need:
1. choose book cover (file should be named after the book);
2. Add name of the book, author, publisher, publication year and description;
3. Choose genre, book type and specify book amount if you chose paper.

Create reports on paper books, paper book orders, download and listen counts

As a Registered user you will be able to...

do everything unregistered user can and...

Order paper books, download electronic and listen to audio ones

As an Administrator user you will be able to...

do everything unregistered user can and...

Issue books

Accept books back

Delete users' accounts (unless they have active orders)
